خطوات التشغيل

  1. تثبيت Node.js:
    إذا لم يكن مثبتاً لديك، حمّله من nodejs.org
  2. فتح Terminal/CMD:
    افتح Terminal أو Command Prompt في مجلد المشروع
    cd trading-calculator-nodejs
  3. تثبيت الحزم:
    npm install
  4. تشغيل التطبيق:
    npm start
  5. فتح المتصفح:
    افتح http://localhost:3000

التحسينات المضافة

  • ✅ أيام العمل: اختيار 5، 6، أو 7 أيام في الأسبوع
  • ✅ المصاريف الشهرية: حساب المصاريف الثابتة
  • ✅ 3 رسوم بيانية: خطي، دائري، وشريطي
  • ✅ 8 بطاقات إحصائية: تفاصيل شاملة عن الأرباح
  • ✅ تصدير CSV/JSON: تصدير البيانات بسهولة
  • ✅ تصميم محسّن: أكثر عصرية واحترافية
  • ✅ API متكامل: للاستخدام البرمجي
