## 🎯 كيفية الاستخدام

### المدخلات الرئيسية
1. رأس المال الأساسي ($)
2. الربح لكل صفقة ($)
3. عدد الصفقات اليومية
4. نسبة إعادة الاستثمار (%)
5. أيام العمل (5-7)
6. المصاريف الشهرية ($)
7. عدد الأيام

### المخرجات
- الرصيد النهائي
- صافي الربح
- نسبة العائد ROI
- إجمالي الأرباح
- السحوبات النقدية
- جداول ورسوم بيانية مفصلة

## 🔧 التخصيص

### تغيير المنفذ
عدّل في `.env`:
```
PORT=3000
```

### تغيير الألوان
عدّل في `public/css/style.css`:
```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
}
```

## 🆘 حل المشاكل الشائعة

### "Port already in use"
```bash

PORT=3001
```

### "Module not found"
```bash
# أعد التثبيت
rm -rf node_modules
npm install
```

- تأكد من تشغيل `npm start`
- افتح Console في المتصفح
- تحقق من عدم وجود أخطاء
